## 🔍 About this Project

To keep it simple: this project aims to simplify the access to MBAACC data directly via discord. The project uses a web scraping tool to gather the data from Mizuumi, which is then customized, cleaned and added with more features in a custom spreadsheet (presented in the repo). It uses Python and some of its libraries (Pandas, BeautifulSoup, requests and Discord.py).


> [!NOTE]
> Considering the scope of this project and the amount of data it deals with, some errors are to be expected. If that happens, please add an issue to this page or contact any of the collaborators via Discord.
